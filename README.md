<h1>Slider Producents Plugin</h1> <p><strong>Opis:</strong> Ta prosta wtyczka dla WordPressa dodaje slider producentów, który można umieścić na stronie za pomocą shortcode. Wtyczka wyświetla logo producentów w formie slajdów z opcjami nawigacji.</p>

<h2>Struktura Wtyczki:</h2> <ul> <li><strong>Informacje o Wtyczce:</strong> Plik zawiera nagłówkowe informacje, takie jak nazwa wtyczki, opis, wersja oraz autor.</li> <li><strong>Funkcja do Załączania Skryptów i Stylów (slider_producents_enqueue_scripts):</strong> Funkcja ta załącza pliki CSS i JavaScript potrzebne do działania slidera.</li> <li><strong>Funkcja Wyświetlania Slidera (slider_producents_shortcode):</strong> Funkcja ta generuje HTML dla slidera producentów i dodaje go za pomocą shortcode'u.</li> <li><strong>Rejestracja Shortcode'u (add_shortcode):</strong> Funkcja ta rejestruje shortcode do wyświetlania slidera producentów.</li> </ul>
